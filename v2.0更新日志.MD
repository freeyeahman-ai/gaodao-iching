## 高岛易断略筮法 更新日志

### ✨ 新增功能

1. **呼吸引导（Breathing Guide）**
   - 起卦前新增可选的三次呼吸引导（吸气/呼气动画），帮助静心凝神
   - 可在 ⚙️ 设置中自由开关，支持一键跳过

2. **天意分堆模式**
   - 除手动滑杆外，新增「天意分堆」（摇筮随机）模式
   - 使用 `crypto.getRandomValues` 加密安全随机源（CSPRNG），模拟真实摇筮过程
   - 筮竹随机摇动后自动落定，拒绝采样消除模偏差

3. **设置中心（Settings）**
   - 新增顶栏 ⚙️ 设置模态框，统一管理：
     - 音效开关（Web Audio API 实时生成筮竹声）
     - 震动反馈开关（移动端 Haptic Feedback）
     - 呼吸引导开关
     - 主题切换：深色（夜色）/ 浅色（纸色）

4. **双主题系统**
   - 支持「夜色」（深色）与「纸色」（浅色）一键切换
   - 自动保存主题偏好至本地存储，刷新后保持

5. **今日起卦提示**
   - 首页显示当日已起卦次数
   - 超过 3 次自动弹出警告（易曰「再三渎，渎则不告」）

6. **结果页爻线动画**
   - 本卦 / 变卦六爻逐条揭示动画，增强仪式感
   - 动爻带红色呼吸灯效果（`box-shadow` 脉冲动画）

7. **动爻爻辞原文**
   - 结果页直接显示动爻对应爻辞原文（如「初九：潜龙勿用」）
   - 附带动爻位置含义解析

8. **分享卡片**
   - 一键生成 900×1280 卦象图片卡片（Canvas 绘制）
   - 包含本卦、变卦、动爻爻辞、卦辞、时间戳等信息
   - 中文避头尾排版，适配移动端保存分享

9. **历史记录增强**
   - 改为模态框展示，不占用结果页空间
   - 支持按「所占之事」或「卦名」实时搜索
   - 支持导出 JSON 格式备份
   - 显示记录统计数量

### 🎨 界面与体验优化

10. **筮竹分堆舞台重构**
    - 49 根虚拟筮竹实时跟随滑杆移动，左右分堆视觉化
    - 受命之手一侧筮竹高亮显示，滑杆拖动带实时音效反馈
    - 滑块使用原生 `input[type="range"]`，触控更流畅

11. **真太阳时显示优化**
    - 独立时间显示卡片，清晰展示北京时间 / 真太阳时 / 地支时辰
    - 显示当前经度、修正分钟数等详细信息
    - 定位失败时优雅降级为北京时间（东经 120° 标准时）

12. **结果页信息架构重构**
    - 卦象解析拆分为「本卦 · 卦辞」与「变卦 · 卦辞」两个独立区块
    - 新增大衍之数计算说明（大衍之数五十，其用四十有九）
    - 整体排版更紧凑，信息层级更清晰

13. **起卦流程引导**
    - 新增步骤进度条（带步骤间连接线），直观显示当前所处阶段
    - 当前问题在起卦过程中置顶显示，防止分心

14. **响应式适配**
    - 移动端按钮自动全宽适配
    - 筮竹视觉、滑杆、卡片在小屏幕下自动缩放

### 🔧 技术升级

15. **音效系统升级**
    - 使用 Web Audio API 替代简单音频播放
    - 滑杆拖动时实时生成筮竹碰撞音效（区分左右手不同音高）
    - 确认分堆时触发组合音效 + 震动反馈

16. **国际化完善**
    - 全面覆盖中英文双语，所有标签、提示、说明、爻辞均支持动态切换
    - 语言偏好自动保存至 `localStorage`
